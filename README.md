 HR Analytics Dashboard — Project 2
An end-to-end HR analytics project that analyzes employee data, identifies attrition
patterns, performs correlation analysis, and presents insights in an interactive dashboard.
---
 Project Structure
```
hr_analytics_dashboard/
├── data/
│   ├── employees.csv          ← Raw employee dataset (100 records)
│   ├── kpis.csv               ← Computed KPI summary (auto-generated)
│   ├── department_summary.csv ← Department-level metrics (auto-generated)
│   └── correlations.csv       ← Feature correlations with attrition (auto-generated)
│
├── scripts/
│   └── hr_analysis.py         ← Full Python analysis pipeline
│
└── dashboard/
    ├── index.html             ← Interactive HTML dashboard (open in browser)
    └── plots/                 ← All chart PNGs (auto-generated by script)
        ├── attrition_by_department.png
        ├── correlation_heatmap.png
        ├── overtime_attrition.png
        ├── salary_distribution.png
        ├── satisfaction_attrition.png
        └── tenure_attrition.png
```
---
 How to Run
Step 1 — Install dependencies
```bash
pip install pandas matplotlib seaborn numpy
```
Step 2 — Run the analysis script
```bash
python scripts/hr_analysis.py
```
This will:
Load and clean the employee dataset
Compute all KPIs
Generate 6 analysis charts → `dashboard/plots/`
Export summary CSVs → `data/`
Print all findings to the terminal
Step 3 — Open the dashboard
```bash
open dashboard/index.html     # macOS
start dashboard/index.html    # Windows
xdg-open dashboard/index.html # Linux
```

Recommendations
Sales Department — Urgent intervention needed. Raise entry-level salaries, reduce
mandatory overtime, and improve work-life balance policies.
Overtime Policy — 81.5% of overtime employees eventually leave. Implement
compensatory time-off or hire additional staff to reduce OT dependency.
New Hire Retention — Employees with 0–2 years tenure are at highest risk.
Strengthen onboarding, mentorship programs, and early career development paths.
Remote / Flexibility — High distance-from-home correlation suggests offering
hybrid work options could significantly reduce attrition.
Compensation Audit — Across all departments, attrited employees earned
~$4,500–$5,000 less per month than retained peers. Salary benchmarking is critical.
